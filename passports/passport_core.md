### 🧠 ПАСПОРТ МОДУЛЯ: main.py (Умный Диспетчер Async)
* **Роль в игре:** Асинхронное ядро проекта. Управляет жизненным циклом бота, запускает сервер и распределяет сообщения.
* **Принцип автоматизации:** При старте сканирует корень проекта, находит все `.py` файлы (кроме системных) и загружает их в память. Если в файле есть метка кнопки, автоматически привязывает её к интерфейсу.
* **Связи с другими модулями:** Запускает веб-сервер через `server.start_hosting()`, берет конфигурацию из `bot_instance.py`, читает карту кнопок из `interface.py`.

### 🤖 ПАСПОРТ МОДУЛЯ: bot_instance.py (Инициализация ИИ-движка)
* **Роль в игре:** Асинхронное сердце проекта на движке Aiogram 3.x. 
* **Конфигурация:** Безопасно забирает токен из переменной среды Render через `os.getenv("BOT_TOKEN")`.
* **Объекты для импорта:** `bot` (отправка сообщений) и `dp` (главный диспетчер).
* **FSM (Память состояний):** Включает `MemoryStorage()`. Хранит игрока в режиме пошагового ввода данных (анкеты).

### 🗛️ ПАСПОРТ МОДУЛЯ: interface.py (Автоматический Интерфейс)
* **Роль в игре:** Слой отображения (UI). Отвечает за динамическую сборку клавиатуры Telegram.
* **Главный инструмент:** Декоратор `@register_game_module(button_text="...")`.
